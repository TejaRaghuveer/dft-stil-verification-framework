# Glossary

- **ATPG**: Automatic Test Pattern Generation.
- **DFT**: Design for Test.
- **STIL**: Standard Test Interface Language (IEEE 1450).
- **TAP**: Test Access Port in JTAG protocol.
- **TDF**: Transition Delay Fault.
- **Coverage Contribution**: Estimated marginal fault coverage gained by a pattern.
- **Pattern Efficiency**: Faults detected relative to pattern cost (size/time/power).
