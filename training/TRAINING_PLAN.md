# Team Training Plan

## 1) Theory Session (1 hour)

Title: **Fundamentals of DFT and Scan Testing**

- DFT basics, scan chain concepts, fault models
- ATPG intent and practical constraints
- How this repository maps to real validation workflows

## 2) Hands-on Session (2 hours)

Title: **Building a DFT Testbench from Scratch**

- Setup and baseline run
- Build a minimal pattern flow
- Add scoreboard and failure diagnostics

## 3) Advanced Session (1.5 hours)

Title: **Pattern Optimization and Coverage Analysis**

- Coverage contribution interpretation
- Compression trade-offs
- ML optimization controls and pitfalls

## 4) Tool Session (1 hour)

Title: **Using Synopsys/Mentor DFT Tools**

- Toolchain mapping to repo outputs
- STIL handoff and ingestion expectations
- Interpreting tool logs and reconciling discrepancies
